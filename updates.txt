## [2026-03-07] Security Hardening (build 40)

### Backend — leaderboard/api.php
- Added HMAC token system: server issues a one-time `{nonce, token}` via `GET ?action=token`; POST submissions must include valid nonce+token signed with `MAGIC_KEY`
- Strict format validation on nonce (32 hex chars) and token (64 hex chars) before any processing — prevents DoS via oversized inputs
- Replay protection: used nonces stored in `used_tokens` table; each token accepted only once
- Nickname raw input capped at 80 bytes before sanitization to prevent regex DoS
- `sanitize_input()` helper: strips non-printable/unicode chars, collapses whitespace
- Rate limiting: max 5 score submissions per IP per minute
- DB error messages no longer leak internal file paths

### Backend — leaderboard/config.php
- `CLEAR_SECRET_KEY` rotated from default `changeme` to a strong random key
- Added `MAGIC_KEY` constant for HMAC signing

## [2026-03-03] Delta Time / Frame-Rate Fix

### Changes
- **game.js**: Introduced `dtf` (normalized delta time, `dt / 16.667`) so all
  physics and movement run at consistent speed regardless of frame rate.
  - Camera movement scaled by `dtf`
  - Enemy AI acceleration, friction, position, and bob animation scaled by `dtf`
  - Player and enemy bullets position and lifetime scaled by `dtf`
  - Invincibility timer and hit flash scaled by `dtf`
  - Friction now uses `Math.pow(factor, dtf)` for correct exponential decay
- **index.html**: Updated GitHub link to point to the actual repo
  (`https://github.com/jadumate/canon-game/`)
